Get Earthly

  • Linux

    • Pre-requisites

    • Docker
    • Git

    Installation

    Terminal
    sudo /bin/sh -c 'wget https://github.com/earthly/earthly/releases/latest/download/earthly-linux-amd64 -O /usr/local/bin/earthly && chmod +x /usr/local/bin/earthly && /usr/local/bin/earthly bootstrap'

    sudo is used for:

    • placing the earthly binary in /usr/local/bin/ and marking it as executable
    • installing auto-completion for your shell
  • Mac

    • Pre-requisites

    • Docker for Mac
    • Git

    Installation

    Terminal
    brew install earthly && sudo earthly bootstrap

    sudo is needed for setting up auto-completion for your shell.

  • Windows (WSL 2)

    • Pre-requisites

    • WSL 2
    • Docker Desktop with WSL2 backend
    • Git

    Installation

    Under wsl, run the following to install earthly.

    Terminal
    sudo /bin/sh -c 'wget https://github.com/earthly/earthly/releases/latest/download/earthly-linux-amd64 -O /usr/local/bin/earthly && chmod +x /usr/local/bin/earthly && /usr/local/bin/earthly bootstrap'

    sudo is used for:

    • placing the earthly binary in /usr/local/bin/ and marking it as executable
    • installing auto-completion for your shell

Verify installation

Terminal
earthly github.com/earthly/hello-world:main+hello

This command executes the target hello from the repository github.com/earthly/hello-world on the branch main.
